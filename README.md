# WasteWise – AI Waste Analyzer & Reuse Marketplace

WasteWise is a web app that uses AI to analyze waste items and suggest reuse options. It also includes a marketplace to buy and sell used items, helping reduce waste and carbon emissions.

## Features

- AI analyzes uploaded images
- Identifies item, materials, and condition
- Gives reuse score and carbon savings
- Suggests reuse ideas (sell, donate, recycle, upcycle)
- Marketplace to list and browse items
- Chat system for buyers and sellers
- User profile with stats and carbon tracking

## Tech Stack

- HTML5
- CSS3
- JavaScript
- Anthropic Claude API
- LocalStorage (no backend)

## How It Works

1. Upload an image or paste a URL
2. AI analyzes the item
3. Get reuse suggestions and carbon impact
4. List item on marketplace or reuse it

## Installation

1. Download the project
2. Open index.html in your browser

## Note

AI analysis requires an API key setup.
