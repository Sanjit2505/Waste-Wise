♻️ WasteWise – AI Waste Analyzer & Reuse Marketplace

WasteWise is a web-based application that uses AI to analyze waste items and suggest sustainable reuse options. It also includes a marketplace where users can buy and sell pre-loved items, helping reduce waste and carbon emissions.

🚀 Features
🔍 AI Waste Analysis
Upload an image or provide an image URL
AI identifies:
Item name & category
Material composition
Condition of the item
Generates:
Reuse score (0–100)
Carbon savings estimate
Reuse & upcycling ideas
♻️ Reuse Recommendations
Suggests multiple ways to reuse items:
Sell
Donate
Upcycle
Recycle
Helps users make eco-friendly decisions
🛒 Marketplace
List items for sale with:
Title, price, description
Category & condition
Browse items listed by other users
Search and filter products by category
View product details and carbon savings
💬 Messaging System
Chat between buyers and sellers
Real-time conversation interface
Manage multiple conversations
👤 User Profile
Track:
Items listed & sold
Messages sent
Total carbon saved
Edit profile details
View environmental impact progress
🌍 Environmental Impact Tracking
Calculates CO₂ saved for each reused item
Displays:
Total carbon savings
Equivalent real-world impact
Encourages sustainable habits
🛠️ Tech Stack
Frontend: HTML5, CSS3, JavaScript
AI Integration: Anthropic Claude API
Storage: Browser LocalStorage (no backend required)
📂 Project Structure
index.html   # Main application file (UI + logic)
⚙️ How It Works
User uploads an image
AI analyzes the item using Claude API
Results include:
Item identification
Reuse score
Carbon savings
User can:
Explore reuse ideas
List item on marketplace
Chat with buyers/sellers
🧠 AI Prompt System

The app sends the image to an AI model with a structured prompt and expects a JSON response containing:

Item details
Condition & reuse score
Carbon impact
Suggested listing data
📦 Installation & Usage
Download or clone the project
Open index.html in your browser
Start using the app

⚠️ Note: AI analysis requires an API integration setup (Anthropic API key)

🔐 Data Storage
All data is stored locally in the browser:
User profile
Marketplace listings
Messages
Carbon data
